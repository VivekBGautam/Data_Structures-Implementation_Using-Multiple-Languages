# Assignment_44

Here contains multiple **C programs** that perform various operations on a **Singly Linear Linked List**.  
These programs focus on **searching, counting, and filtering elements** in the list, helping to understand traversal and conditional logic in linked lists.

---

## Programs Overview

- **program44_1.c** – Searches for a given element in the linked list and returns `true` if it exists.  
  **Function Prototype:** `bool Search(PNODE Head, int No);`

- **program44_2.c** – Counts the number of nodes containing **even values**.  
  **Function Prototype:** `int CountEven(PNODE Head);`

- **program44_3.c** – Counts the number of nodes containing **odd values**.  
  **Function Prototype:** `int CountOdd(PNODE Head);`

- **program44_4.c** – Counts the **frequency** of a given number in the linked list.  
  **Function Prototype:** `int Frequency(PNODE Head, int No);`

- **program44_5.c** – Displays **only the even elements** of the linked list.  
  **Function Prototype:** `void DisplayEven(PNODE Head);`
